A fully interactive dashboard built using Power BI to analyze retail sales data. This project highlights business insights such as total sales, profit trends, regional performance, and product category comparisons.

🔍 Key Features

- Sales, profit, and discount analysis
- Region-wise and category-wise breakdown
- Time-based trends using slicers and filters
- KPIs and conditional formatting for better insights

🛠 Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
